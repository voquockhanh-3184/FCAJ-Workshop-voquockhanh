1. Open **IAM Console** (Click search bar -> Search keyword **IAM** -> Select **IAM**)

![IAM Console](/images/5-Workshop/5.2-IAM/5.2.2-create-account-alias/IAM1.1.png)

2. On the IAM interface (IAM Dashboard), look for the **Account Alias** section in the **AWS Account** area -> Click **Create**

![Create Account Alias](/images/5-Workshop/5.2-IAM/5.2.2-create-account-alias/IAM1.2.png)

![Create Account Alias Dialog](/images/5-Workshop/5.2-IAM/5.2.2-create-account-alias/IAM1.3.png)

3. In the **AWS Account** section, if you see the Account Alias + URL link, it means the creation was successful.

![Account Alias Success](/images/5-Workshop/5.2-IAM/5.2.2-create-account-alias/IAM1.4.png)
