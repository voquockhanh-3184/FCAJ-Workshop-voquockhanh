# Examora - Online Exam Platform 

#### Overview

**Examora** is a serverless online exam platform designed to handle scalable quiz generation, grading, and secure asset management.

In this workshop, you will learn how to design and build a secure, scalable, and resilient cloud architecture on AWS using console configurations and infrastructure as code basics.
